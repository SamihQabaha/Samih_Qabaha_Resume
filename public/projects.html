Samih Qabaha Projects

Most relevant projects as Freelancer:

  • October 2022 – March 2023: Show game system.

    • Design and implementation of Servers hardware (Raspberry Pis), addressable RGBs and sound signaling, LCDs control, buttons, on stage local communications between different hardware parts and LCDs, and tablets, as well as integration and contest structure and flow.
  • November 2020: Smart bedroom.
    • Concept model for a furniture company to develop a bedroom with smart features and home automation build-in.
    • My role was hardware design and implementation, including lights, tele-switches and relays, LCD dashboard, cloud platform setup with app, wi-fi and MQTT communication programing, music player scripting, programing voice commands and speech recognition, integration google assistant with the system.
  • May 2020: Standalone Face and mask recognition for covid-19 prevention.
    • Development and training of a Deep learning model using opensource dataset of images to detect faces and masks with very high accuracy and speed Using OpenCV, Keras-TensorFlow and implemented on Raspberry pi computer with webcam.
  • June 2019: Face recognition and environment description using AWS APIs on Raspberry Pi.
    • A system implemented on Raspberry Pi minicomputer with USB camera, that use AWS in the back end to detect faces and recognize them in the image, and describe the image by text.
  • March 2019: Standalone Optimized face detection model and recognition system for use on Raspberry Pi.
    • Using OpenCV library to detect faces and then recognizes the pretrained faces to identify people and add them to a database, similar to an authentication system using face ID. Using local computing resources of small computer equipped with camera.
  • October 2018 – February 2019: IoT enable cloud Platform for controlling and monitoring of liquid Tanks levels.
    • A GSM-IoT nodes for local telecom operator that can read level of liquid in tanks and send this data to the cloud, and then perform some actions.
    • I was responsible for designing and developing of the circuit electronics including PCB and using ESP32 microcontroller and GSM module.
    • Programing the necessary APIs to connect GSM-module with ESP32 and for communication over GSM modules to the cloud as well as programing the ESP32 to work with level sensor.
  • July – September 2018: Smart Lawn Robot.
    • A robot to take care of lawns detect yellow or sick grass regions in the garden then take care of this regions.
    • I was responsible for creation of data set of images, train the ML model, implement the model on raspberry pi equipped with camera to take pictures and classify the ground, detect grass, and check if it is normal or abnormal.
    • Development of local Positioning System in the garden using stereo cameras to construct depth maps and 3D coordinate system, and pass it to robot’s raspberry pi.
    • Develop the wi-fi communications between the robot and positioning system Pis. all using python OpenCV, raspberry Pis, and cameras.
  • July 2018: Home Automation Device.
    • Home Automaton project to fully produce local Palestinian control devices including Smart electricity switches and outlets to control home over the Internet.
    • I was responsible for design the needed electronics and PCBs, programing microcontroller to take sensor readings and control solid-state relays and power electronics, as well as communicating with cloud server using wi-fi and MQTT protocol. using C++, ESP32 microcontroller.
  • June 2016: Graduation project, “Smart Visually impaired Aid” (SVIA). Arab American University Jenin.
    • A Complete and integrated system build using ML algorithms implemented on Matlab, the system was concept model for a device that used by visually impaired people in which it help them in mobility using 3D tone signals system, reading texts, recognize traffic signs around, and finally search for their lost belongings like their phone or keys by 3D virtual sound generation based on depth images.
    • The system uses stereo cameras to construct 3D map of the environment then guide the impaired person by 3D tone sound modulate the space. As well as ML models to detect signs and make OCR.
